# Setup Instructions

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Virtual Environment

### On Linux/Mac:
```bash
source venv/bin/activate
```

### On Windows:
```bash
venv\Scripts\activate
```

## Install Kortex

```bash
pip install ./kortex_api-2.7.0.post5-py3-none-any.whl
```

## Deactivate (when done)

```bash
deactivate
```
